## Project overview
This is a small frontend project built for **learning and experimentation**.  
The main goal was to understand **website layouts, responsive design, and CSS styling techniques** by building a realistic food delivery landing page.

The project focuses on:
- Responsive layouts using **CSS media queries**
- Clean and semantic **HTML structure**
- Styling with **Flexbox, Grid, colors, and typography**
- Designing a real-world landing page UI

---

## Key features
- Responsive header and navigation
- Hero/banner section showcasing featured dishes
- Menu sections with food cards
- Call-to-action section for orders or contact
- Mobile-first layout with breakpoints for tablet and desktop

---

## Technologies used
- **HTML5**
- **CSS3** (Flexbox, Grid, Media Queries)
- No JavaScript (static prototype, easy to extend later)
